# 🕹️ Игра Maze (Лабиринт)

Классическая 2D-аркада, написанная на Python с использованием библиотеки Pygame. Игроку предстоит управлять персонажем, избегать препятствий и врагов, чтобы добраться до сокровища.

---

## 🚀 Особенности игры
* **Управление**: Движение стрелочками на клавиатуре.
* **Звуковое сопровождение**: Фоновая музыка во время игры.
* **Препятствия**: Стены лабиринта, при столкновении с которыми игра заканчивается.
* **Враги**: Двигающийся киборг-монстр, патрулирующий территорию.
* **Условие победы**: Добраться до звезды.
